Device List

You can see all the devices,
plus you can delete them and you can see in detail each device.

Person List

You can see all the persons,
plus you can delete them, add one and you can see in detail each person.
If you see a person with a house in detail you can click to see the house's detail.

Home List

You can see all the homes,
plus you can delete them, add one and you can see in detail each home.
You can also add SmartDevice to a home if you see this home in detail
